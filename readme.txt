大二的时候和班上一个同学参加学院举办的程序设计竞赛时用C#写的程序
题目要求
	1.在一个停车场内，有连续三个停车位，1、3号车位都有车，利用程序模拟把泊车起始区域中任意位置的车停到2号车位的过程
	2.实时记录车四个轮子的朝向，总共的行驶距离以及泊车总耗时，最终输出到txt文件中
	3.车位长度：1.5倍车长

个人认为程序的亮点在泊车的算法。
我和搭档将此算法分成了两步实现：
1.在泊车起始区域中找到一个固定区域A（四个点围成的一个矩形），并推导出A停靠至二号车位的固定算法
2.实现将泊车起始区域内任意位置的车移动到A的算法
将1算法和2算法合并，即得到了任意位置泊车到二号车位的算法。
